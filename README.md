Site Currículo
---
Este é o repositório de um Site Currículo, um projeto desenvolvido para apresentar habilidades, experiências, projetos e formas de contato de maneira moderna, responsiva e visualmente atraente.

O site é totalmente feito com HTML, CSS, React, Tailwind e JavaScript, focando em animações suaves, design minimalista e boa leitura em qualquer dispositivo.

Funcionalidades
---
Página inicial com apresentação pessoal

Sessão sobre mim

Lista de habilidades e tecnologias utilizadas

Portfólio de projetos

Sessão de contato

Background animado e responsivo

Como Rodar Localmente
--
Clone o repositório: 
git clone https://github.com/Igudevkit/Site-Curriculo

Entre na pasta: 
cd Site-Curriculo

Instale dependências: 
npm install

Inicie o servidor: 
npm run dev
